Welcome to Flash!

This is a secure messaging app built with Flask.

To get started, please register or log in using the links above.

About Flash

Flash is designed to provide secure messaging between users. It uses RSA and AES encryption to ensure that your messages remain private and secure.

How It Works

  • Registration: When you register, RSA keys (public and private) are generated for you and stored securely.
  • Sending Messages: When you send a message, it is encrypted using AES encryption, and the AES key is further encrypted using the recipient's public RSA key.
  • Receiving Messages: When you receive a message, your private RSA key is used to decrypt the AES key, which is then used to decrypt the message content.
